실행결과: setTitle은//주석이라 제목은 예3 버튼은 1,2,3 3개이지만 마지막 3번의 버튼만 보인다.

import java.awt.*; class 실습06{
public static void main(String[] args) {
Frame f= new Frame("예 3");
FlowLayout f1 = new FlowLayout();
f.setLayout(f1);
Button b1= new Button("OK");
Button b2= new Button("CANCEL");
Button b3= new Button("stop");
f.add(b1); f.add(b2); f.add(b3);
f.setSize(300,150);
f.setVisible(true);
} }

실행결과: 제목은 예 3 버튼은 flowLayout이 없으면 마지막 b3(stop)만 있겠지만 flowlayout이 있으므로 b1 b2 b3가 다 나온다

import java.awt.*;
public class 실습07_1 {
public static void main(String[] args) {
Frame fr= new Frame("플로우레이아웃데모");
fr.setLayout(new FlowLayout(FlowLayout.LEFT,30,0));
fr.add(new Button("버튼1"));
fr.add(new Button("버튼2"));
fr.add(new Button("버튼3"));
fr.add(new Button("버튼4"));
fr.add(new Button("버튼5"));
fr.add(new Button("버튼6"));
fr.add(new Button("버튼7"));
fr.add(new Button("버튼8"));
fr.add(new Button("버튼9"));
fr.setSize(300,150); fr.setVisible(true); }
}

실행결과: 제목 플로우레이아웃데모 수평간격30 만큼 정렬한 버튼 1~9번 까지 나온다
